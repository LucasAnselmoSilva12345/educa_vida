Ensino+

O Ensino+ (Ensino plus) é o nome do tão citado projeto de ajuda para os alunos, desenvolvido por grandes profissionais da educação brasileira.

Com uma base educacional enorme, com foco em alunos de 6° Ano até 3° Ano do Médio.

Nessa base, além de muitos estudos, os alunos também são submetidos a testes práticos para avaliação de ensino, e testes de desenvolvimento de carreira.
