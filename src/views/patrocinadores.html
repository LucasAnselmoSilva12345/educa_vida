MEC

Somos reconhecidos pelo Ministério da Educação, como um programa de auxÍlio para as escolas públicas, com vários certificados registrados no portal do MEC

MEC

ETEC

Temos uma parceira com a Escola Técnica Estadual, onde levamos nossos alunos para conhecer o dia-a-dia dos alunos dessa escola, e mostrar sobre os cursos técnicos de cada instituição, para motivar nossos alunos a entrarem num colégio técnico.

ETEC

IFSP

Assim como a ETEC, também temos uma parceria com o Instituto Federal de Educação, Ciência e Tecnologia de São Paulo, onde levamos nossos alunos para conhecer o dia-a-dia dos alunos dessa escola, e mostrar sobre os cursos técnicos de cada instituição, para motivar nossos alunos a entrarem num colégio técnico.

IFSP

Lego

Uma parceria com a empresa Lego, para implementação das aula de robótica, nas escolas, para treinamento de montagem e lógica em nossos alunos

Lego

Positivo

Técnologia é uma coisa fundamental hoje em dia, e para ter acesso a ree, nada melhor que os computadores positivo, temos uma parceria para obtermos os melhores computadores, para instalarmos nas escolas, para auxiliar os alunos nas aulas.

Positivo

Faber Castell

Uma parceira para linha de Materias de otima qualidade? Temos!!, a Faber Castell é nossa aliada em garantir otimos materiais para nossos alunos, para ajuda-lôs nos estudos.

Faber Castell

Ceunsp

A Ceunsp é responsavel pelos modelos de faculdades em nosso projeto, uma parceria para mostrar o dia-a-dia de um aluno de faculdade, e realizar a sensação de interesse de nossos alunos à entrarem nas faculdades.
